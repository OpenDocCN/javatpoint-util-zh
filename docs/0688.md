# 乳胶数学符号

> 原文:[https://www.javatpoint.com/latex-math-symbols](https://www.javatpoint.com/latex-math-symbols)

数学或数学符号是那些用作中间体来执行数学功能的符号。数学中的变量和函数有多种用途。

建议使用命令$作为数学模式。下面提到的输入命令被插入到$之间以产生相应的输出。

最常用的数学类型命令如下:

| 输入命令 | 输出(数学模式) | 输入命令 | 输出(数学模式) |
| `\&#124;` | `‖` | `\leq` | `≤` |
| `\alpha` | `α` | `\lnot` | `¬` |
| `\ast` | `*` | `\mid` | `&#124;` |
| `\bigodot` | `ʘ` | `\mu` | `µ` |
| `\approx` | `≈` | `\pm` | `±` |
| `\bullet` | `●` | `\P` | `¶` |
| `\copyright` | `©` | `\div` | `÷` |
| `\Box` | `□` | `\prod` | `∏` |
| `\downarrow` | `↓` | `\perp` | `ꓕ` |
| `\delta` | `∆` | `\sqrt` | `√` |
| `\geq` | `≥` | `\int` | `∫` |
| `\gets` | `←` | `\oe` | `œ` |
| `\gg` | `»` | `\mho` | `Ʊ` |
| `\infty` | `∞` | `\deg` | `deg` |
