我们需要以 **sudo 用户**的身份输入以下命令，以便从系统中卸载**团队查看器**，而不删除我们可能已经进行的任何配置:


系统会提示我们一个 ***是/否*** 选项开始卸载程序。我们需要输入 ***Y*** ，点击回车开始。然后， ***TeamViewer*** 将从我们的系统中删除。
